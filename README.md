# my_vimrc
just a place to store my vimrc

requires:
Pip  
cmake (via pip install)  
brew

# Steps:
- install vim
- install nvim  
- install pynvim https://github.com/neovim/pynvim 
- install vundle  
- move init.vim to ~/.config/nvim/init.vim  
- move .vimrc to ~/.vimrc
- nvim ~/.vimrc -- :PluginInstall
